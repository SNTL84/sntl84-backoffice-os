## 🔴 Problem

Small businesses and founders in India manage HR through **3–5 separate spreadsheets** — attendance in one file, payroll in another, petty cash in a third, field visits in WhatsApp messages. Every month, reconciling these takes **6+ hours of admin time** and still produces errors. There's no single view of what's happening with the team.

---

## 🛠️ What I Built

**BACKOFFICE OS v2.0** — a single-file, zero-dependency HTML dashboard that replaces all of it:

- **Attendance Tracking** — Clock In/Out, break deduction, live status
- **Leave Management** — Apply, approve, reject with leave wallet per employee
- **Payroll** — Salary config, auto-calculate, payslip generation, payout queue
- **Expense Claims** — Submit, review, and approve with audit trail
- **Field Visit Logs** — Location, mode, hours, and expenses for on-ground staff
- **Petty Cash Ledger** — Float tracking, category breakdown, voucher log
- **HR Admin Panel** — Employee management, payroll cycles, leave policy config

No install. No database. Open the file in a browser and it works.

```bash
# That's it:
open SNTL-84-hr_backoffice.html
```

# 🏢 BACKOFFICE OS v2.0 — HR Management Suite

> **I Automate What's Costing You Money.**

A zero-dependency, single-file HTML utility tool for small teams and founders to manage their entire back-office HR workflow — built with precision by **Milan · SNTL 84**.

---

## 🚀 Features

| Module | Description |
|--------|-------------|
| 📊 Dashboard Overview | Real-time KPIs — headcount, attendance, payout & petty cash |
| 🕐 Attendance Tracking | Clock In / Clock Out, break deduction, live status |
| 📅 Leave Management | Leave wallet, apply / approve / reject requests |
| 💰 Payroll | Salary config, payslip generation, payout approval queue |
| 🧾 Expense Claims | Submit, review & approve employee expense claims |
| 📍 Field Visit Logs | Location, mode, hours & expense tracking for field staff |
| 🏦 Petty Cash Ledger | Float tracking, category breakdown, voucher log |
| ⚙ HR Admin Panel | Employee management, payroll cycles, leave policy config |
